## 디렉토리 구조
```
src
 ├── backend
 |       ├── auth           // 인증 서버
 |       ├── chat           // 채팅 서버
 |       ├── community      // 커뮤니티 서버
 |       ├── notification   // 알림 서버
 |       ├── presence       // 상태관리 서버
 |       └── signaling      // 시그널링 서버
 |      
 ├── frontend
 |       ├── web            // 웹 애플리케이션 (Vue)
 |       └── ios            // 모바일 애플리케이션 (iOS)
 |      
 └── infrastructure
         └── gateway        // 게이트웨이
```
